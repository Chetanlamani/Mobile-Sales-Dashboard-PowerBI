# 📊 Mobile Sales Dashboard | Power BI

An interactive **Power BI dashboard** developed to analyze mobile sales performance using dynamic visualizations, KPIs, and business insights. This project demonstrates data cleaning, modeling, DAX calculations, and dashboard design to support data-driven decision-making.

---

## 📌 Project Description

The Mobile Sales Dashboard provides a comprehensive view of sales performance across different brands, regions, products, and time periods. It enables users to monitor key metrics, identify trends, and gain actionable insights through interactive filters and visualizations.

---

## 🚀 Features

- 📱 Mobile Sales Analysis
- 💰 Revenue & Profit KPIs
- 📈 Monthly Sales Trends
- 🌍 Region-wise Performance
- 🏆 Brand-wise Sales Analysis
- 👥 Customer Insights
- 📊 Interactive Slicers & Filters
- ⚡ Dynamic DAX Measures

---
